스마트 컨트랙트

-서면으로 이루어지던 계약을 코드로 구현하는 특정 조건이 충족되었을 때 해당 계약이 이행되도록 하는 개념

블록체인에서 동작하는 응용프로그램의 단위로 스마트 컨트랙트의 흐름은 웹 응용프로그램 개발과 유사함

블록체인의 서비스를 정의하여 프로세스를 수행하기 위한 프로그램 으로 데이터를 만들어 낸다
블록체인 네트워크 위에서 동장하는 프로그램! => 블록 DATA를 만들어 낸다
이러한 것을 비지니스, 계약, 암호 등으로 풀어내는 것들이 필요하다

모든 프로그램은 다른 프로그램에 입력값이 될 수 있다 !!

스마트 컨트랙트 개발
스마트 컨트랙트 배포


스마트 컨트랙트의 취약성 => 이더리움 DAO
public 스마트 컨트랙트는 취약성 때문에 결정적 언어를 사용 -> 결과 예측 가능(Solidity)
private 스마트 컨트랙트는 비결정적을 사용 -> Go, Java, 등 취약점이 높아도 책임을 물을 수 있기에 생산성 높은 것을 사용

스마트 계약으로 인해 중계자 없이 다양한 거래를 실현(암호화폐)

- node.js ( 웹 라이브러리 ), spring boot
